# Skill: 利润质量验证（财报透视 — Module 3）

**知识来源**：《一本书读懂财报》肖星著，清华大学经管学院教授，2019年修订版
**案例库**：`references/case_library.md`（所有 CASE-M3-XXX 原文引用均来自此文件）
**独立使用**：本 skill 可单独调用，也可接收 M1 行业基准报告后作为编排流程第三步

## 调用时机

以下任一情况可单独调用本 skill：
- 用户想判断某公司的利润是否真实、可持续
- 用户发现净利润大增但主营业务未明显改善
- 用户担心公司通过非经常性损益、赊销、存货操纵等手段虚增利润
- 作为完整财报分析的第三步（建议先运行 M1 行业背景分析）

**若未提供 M1 行业基准报告**：执行时自行询问行业类型，判断行业正常毛利率和应收账款回收周期。

---

## 模块输入

- M1 行业坐标基准报告（行业毛利率区间、应收账款正常水平）
- 用户提供的利润表数据：营业收入、营业成本、毛利、期间费用、营业利润、营业外收入/支出、净利润
- 资产负债表辅助数据：应收账款、存货（近3年最佳）
- 现金流量表：经营活动现金流

## 模块输出

向 orchestrator 返回「利润质量报告」，包含：
- 利润各层次的质量评级（🔴风险 / 🟡关注 / 🟢正常）
- 可持续盈利能力判断
- 利润与现金流的匹配度评估
- 需用户进一步核实的问题

---

## 执行流程

### 总览：利润质量验证的五个步骤

```
利润表（从上到下）
├── Step 1：毛利率趋势      ← 主业定价权与成本控制
├── Step 2：费用侵蚀程度    ← 期间费用是否合理
├── Step 3：非经常性损益    ← 剔除一次性项目后的主业盈利
├── Step 4：利润现金含量    ← 利润有没有变成真实现金
└── Step 5：ROIC vs WACC   ← 经济利润视角（可选，数据充分时执行）
```

每步执行：计算指标 → 与 M1 行业区间对比 → 触发红旗则引用对应案例 → 记录评级

---

### Step 1：毛利率趋势——主业核心盈利能力

**核心问题**：毛利率是在提升、稳定还是下滑？是否与行业竞争环境一致？

**计算指标**：
```
毛利率 = (营业收入 - 营业成本) / 营业收入
连续3年毛利率趋势
毛利率 vs 行业均值
```

**判断规则**：

| 信号强度 | 条件 | 评级 |
|---------|------|------|
| 🟢 正常 | 毛利率稳定或上升，且与行业水平一致 | 主业盈利健康 |
| 🟡 关注 | 毛利率缓慢下滑，但仍高于行业均值 | 竞争压力加大 |
| 🔴 红旗 | 毛利率快速下滑，或远低于同行 | 定价权丧失或成本失控 |

**追问用户**：
- 毛利率下滑是因为降价还是成本上升？
- 竞争对手毛利率是否同步下滑（行业性问题）？

---

### Step 2：期间费用侵蚀——费用控制能力

**核心问题**：销售/管理/财务费用的增速是否快于营收增速？

**计算指标**：
```
期间费用率 = (销售费用 + 管理费用 + 财务费用) / 营业收入
营业利润率 = 营业利润 / 营业收入
费用增速 vs 营收增速（连续年度对比）
```

**判断规则**：

| 信号强度 | 条件 | 评级 |
|---------|------|------|
| 🟢 正常 | 费用增速 ≤ 营收增速，期间费用率稳定 | 费用控制良好 |
| 🟡 关注 | 费用增速略超营收增速，但毛利率可吸收 | 关注效率 |
| 🔴 红旗 | 费用增速明显超过营收增速，营业利润率持续压缩 | 费用失控 |

---

### Step 3：非经常性损益——剥离一次性项目

**核心问题**：净利润中有多少来自可持续的营业利润，有多少来自不可持续的营业外收入？

**计算指标**：
```
营业外收入占净利润比 = 营业外收入 / 净利润
扣非净利润 = 净利润 - 非经常性损益（含营业外收入、政府补贴等）
扣非净利润增速 vs 净利润增速
```

**判断规则**：

| 信号强度 | 条件 | 评级 |
|---------|------|------|
| 🟢 正常 | 营业外收入占比 < 10%，扣非净利润与净利润接近 | 利润来源可持续 |
| 🟡 关注 | 营业外收入占比 10%-30% | 存在一次性贡献，持续性存疑 |
| 🔴 红旗 | 营业外收入占净利润 > 30%，或扣非净利润大幅低于净利润 | 主业赚钱能力不足 |

**红旗触发时 → 引用 CASE-M3-001**：
> 两家公司净利润一样，投资哪家？要看"哪里赚来的钱"——
> 900万来自营业利润的企业，比900万来自卖设备补贴的企业，未来的持续盈利能力强得多。

---

### Step 4：利润现金含量——利润有没有变成真实现金

**核心问题**：赚到的利润是否真实流入公司账户，还是积压在应收账款和存货里？

**计算指标**：
```
现金含量 = 经营活动现金流 / 净利润（理想值 ≥ 1.0，警惕值 < 0.8）
应收账款增速 vs 营收增速
存货 → 应收账款异动（存货减少同期应收账款大增 = 赊销转移风险）
```

**判断规则**：

| 信号强度 | 条件 | 评级 |
|---------|------|------|
| 🟢 正常 | 现金含量 ≥ 0.8，应收账款增速 ≤ 营收增速 | 利润真实回收 |
| 🟡 关注 | 现金含量 0.5-0.8，或应收账款增速略超营收 | 回款压力，需关注趋势 |
| 🔴 红旗 | 现金含量 < 0.5 或经营现金流为负；应收账款增速持续超过营收增速 | 利润质量极差 |

**存货减少+应收账款大增时 → 引用 CASE-M3-002**：
> 存货变应收账款，看起来货卖掉了，但关键是钱有没有收回来。
> 造纸企业用赊销消灭存货，带来14%应收账款占比——若收不回就全是坏账。

**利润正但经营现金流为负时 → 引用 CASE-M3-004 + CASE-M3-005**：
> A公司（电子书阅读器）vs B公司（通信设备）：同样"利润正现金流负"，
> 结局却截然不同——关键在于应收账款背后的客户是否会赖账。

---

### Step 5：ROIC vs WACC——经济利润视角（数据充分时执行）

**核心问题**：净利润为正，但ROIC是否高于资本成本（WACC）？企业是否真正在为股东创造价值？

**计算指标**（需要有息负债、股东权益、行业平均盈利水平数据）：
```
投资资本 = 有息负债 + 股东权益
ROIC = 税后营业净利润 / 投资资本
WACC = 有息负债/投资资本 × 债务成本 × (1-税率) + 股东权益/投资资本 × 行业平均ROE
经济利润 = (ROIC - WACC) × 投资资本
```

**判断规则**：

| 信号强度 | 条件 | 评级 |
|---------|------|------|
| 🟢 赚钱 | ROIC > WACC，经济利润为正 | 真正为股东创造价值 |
| 🟡 及格 | ROIC ≈ WACC，经济利润接近零 | 勉强覆盖资本成本 |
| 🔴 亏损 | ROIC < WACC，经济利润为负 | 账面盈利但实际销毁股东财富 |

**ROIC < WACC 时 → 引用 CASE-M3-003**：
> 净利润为正只是会计的及格线，不代表经济上真正赚钱。
> 刘洋公司2017-2018年账面盈利，但ROIC低于WACC——经济利润为负，在消耗股东财富。
