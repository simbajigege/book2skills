# Skill: 周转率与偿债能力（财报透视 — Module 5）

**知识来源**：《一本书读懂财报》肖星著，清华大学经管学院教授，2019年修订版
**案例库**：`references/case_library.md`（所有 CASE-M5-XXX 原文引用均来自此文件）
**独立使用**：本 skill 可单独调用，也可接收前序模块报告作为编排流程第五步

## 调用时机

以下任一情况可单独调用本 skill：
- 用户想判断企业债务风险是否过高
- 用户想了解应收账款/存货周转是否正常
- 用户想分析短期或长期偿债能力
- 用户发现周转率大幅改善，想验证真实性

---

## 模块输入

- M1 行业基准报告（行业典型周转率、资产负债率区间）
- 用户提供的资产负债表 + 利润表数据（最好3年）：
  - 流动资产、流动负债、存货、短期借款
  - 应收账款、固定资产、总资产
  - 营业收入、营业成本、净利润、财务费用、所得税
  - 总负债

## 模块输出

向 orchestrator 返回「偿债能力与营运效率报告」，包含：
- 短期/长期偿债能力评级
- 各项资产周转率及异常信号
- 周转率改善真实性验证

---

## 执行流程

### Step 1：短期偿债能力

**核心问题**：企业能否在短期内偿还流动负债？

**计算指标**：
```
流动比率 = 流动资产 / 流动负债
速动比率 = (流动资产 - 存货) / 流动负债
调整后流动比率 = 流动资产 / (流动负债 - 短期借款)  ← 中国企业专用
```

**判断规则（中国企业标准）**：

| 信号强度 | 流动比率 | 调整后流动比率 | 评级 |
|---------|---------|--------------|------|
| 🟢 健康 | 1-2 | ≥ 2 | 短期偿债无虞 |
| 🟡 关注 | < 1 | 1-2 | 需核查短贷续借能力 |
| 🔴 风险 | < 0.5 且调整后仍低 | < 1 | 短期偿债压力大 |

**流动比率偏低时 → 引用 CASE-M5-001**：
> 流动比率2在中美两国含义完全不同：美国破产公司破产前恰好是2；
> 中国企业正常水平就在1-2，因为短期借款可以借新还旧。
> 分析中国企业时，应剔除短期借款后再算流动比率。

**存货占比大时 → 引用 CASE-M5-002**：
> 存货是流动资产中最难变现的——要先卖掉，再等钱收回来。
> 存货占比高的行业，速动比率才是真正的安全垫。

---

### Step 2：长期偿债能力

**核心问题**：企业利息覆盖能力如何？整体负债水平是否合理？

**计算指标**：
```
息税前收益（EBIT）= 净利润 + 所得税 + 财务费用
利息保障倍数 = EBIT / 财务费用
资产负债率 = 总负债 / 总资产
```

**判断规则**：

| 指标 | 健康 | 关注 | 风险 |
|------|------|------|------|
| 利息保障倍数 | ≥ 5倍 | 3-5倍 | < 3倍 |
| 资产负债率（一般行业） | < 50% | 50%-70% | > 70% |
| 资产负债率（重资产行业） | < 70% | 70%-85% | > 85% |

**利息保障倍数不足时 → 引用 CASE-M5-003**：
> 判断利息偿还能力要用EBIT（净利润+所得税+利息），而非净利润。
> 案例中5.8倍意味着赚的钱够还5.8次利息；低于3倍需要警惕。

**资产负债率偏高时 → 引用 CASE-M5-004**：
> 资产负债率80%的航空公司比比皆是，重资产行业天然高负债。
> 判断时必须对行业：中国上市公司均值约45%，重资产行业可到70-80%。

---

### Step 3：各项资产周转率

**核心问题**：资产是否被高效利用？周转率是否与行业一致？

**计算指标**：
```
应收账款周转率 = 营业收入 / 应收账款（理想：收款周期 < 90天）
存货周转率 = 营业成本 / 存货（行业差异大）
固定资产周转率 = 营业收入 / 固定资产
总资产周转率 = 营业收入 / 总资产
```

**判断规则**：与 M1 行业基准对比，重点关注以下异常：
- 应收账款周转率下降 = 回款周期拉长，赊销风险上升
- 存货周转率下降 = 产品积压，市场竞争力下降
- 固定资产周转率下降 = 产能过剩或设备闲置

---

### Step 4：周转率改善真实性验证

**核心问题**：周转率改善是真实经营改善，还是坏账计提/关联方资产剥离的假象？

**核查步骤**：
1. 对比当期竞争环境：行业是否真的好转？
2. 查看应收账款/存货减少的原因：是正常回款销售，还是计提坏账或关联交易？
3. 核查利润：若周转率改善同时利润大幅下滑，可能是资产减值导致

**周转率异常改善时 → 引用 CASE-M5-005**：
> 彩电企业2012年周转率"咸鱼翻身"：原来是22亿坏账计提+把资产打包卖给关联企业。
> 周转率改善不等于经营改善——几年后的几十亿亏损才是真相。
> 应收账款和存货减少，必须查清是真实回款还是减值与转移。
