# Skill: 好公司综合评定（财报透视 — Module 6）

**知识来源**：《一本书读懂财报》肖星著，清华大学经管学院教授，2019年修订版
**案例库**：`references/case_library.md`（所有 CASE-M6-XXX 原文引用均来自此文件）
**独立使用**：本 skill 通常作为最终综合评定，需汇总前序模块输出

## 调用时机

以下任一情况调用本 skill：
- 用户想综合评估企业是否是"好公司"
- 用户想判断企业战略定位是否清晰、执行是否有效
- 用户想了解ROE的驱动因素（效益/效率/杠杆）
- 用户想用经济利润框架评判企业是否真正赚钱

---

## 模块输入

- M1-M5 各模块输出报告（汇总分析结论）
- 用户提供的关键财务比率：毛利率、总资产周转率、ROE、ROA
- 可选：有息负债、行业平均ROE（用于计算WACC和经济利润）

## 模块输出

向 orchestrator 返回「好公司综合评定报告」，包含：
- 战略定位判断（差异化 vs 成本领先）
- 战略执行效果评级
- 杜邦分析ROE驱动因素拆解
- 经济利润视角的最终判断

---

## 执行流程

### Step 1：战略定位识别

**核心问题**：企业的战略定位是差异化还是成本领先？财务特征与战略是否匹配？

**判断框架**（来自 CASE-M6-003）：

```
差异化战略 → 高毛利率 + 低总资产周转率（苹果、卷烟纸）
成本领先战略 → 低毛利率 + 高总资产周转率（华为、新闻纸）
两者兼顾 → 极少见，护城河极强的企业（如早期苹果）
执行失败 → 毛利率低 + 周转率也低（战略舍弃了方面A，但未换来方面B）
```

**输出示例**：
> 该企业毛利率35%（高），总资产周转率0.5次/年（低）→ 差异化战略定位，财务特征匹配。

**引用 CASE-M6-001（苹果 vs 华为）或 CASE-M6-003（卷烟纸 vs 新闻纸）**。

---

### Step 2：战略执行效果评级

**核心问题**：战略是否被有效执行？ROA是否达到行业竞争力水平？

**计算指标**：
```
总资产报酬率（ROA）= 净利润率 × 总资产周转率
ROA vs 行业平均ROA
```

**判断规则**：

| 信号强度 | 条件 | 评级 |
|---------|------|------|
| 🟢 执行有效 | ROA ≥ 行业平均，战略财务特征吻合 | 战略执行良好 |
| 🟡 执行一般 | ROA ≈ 行业平均，但某一维度（毛利或周转）明显不足 | 执行存在短板 |
| 🔴 执行失败 | 毛利率和周转率均低于行业，ROA 落后 | 战略未能落地 |

**ROA 低于行业时 → 引用 CASE-M6-003**：
> 战略是策略性地舍弃一个方面去追求另一个方面。
> 如果舍弃了该舍弃的，但没换来想要的，说明战略执行出了问题。

---

### Step 3：杜邦分析——ROE驱动因素拆解

**核心问题**：股东回报（ROE）的来源是效益、效率，还是杠杆？

**计算指标**（来自 CASE-M6-002）：
```
ROE = 净利润率 × 总资产周转率 × 权益乘数
权益乘数 = 总资产 / 股东权益 = 1 / (1 - 资产负债率)
ROA = 净利润率 × 总资产周转率
ROE = ROA × 权益乘数
```

**驱动因素判断**：

| ROE来源 | 可持续性 | 评级 |
|--------|---------|------|
| ROA高（效益+效率） | 可持续，竞争力强 | 🟢 |
| ROA一般但权益乘数高（高杠杆） | 不可持续，且风险放大 | 🟡 |
| ROA低且权益乘数高 | 危险组合：盈利差+债务重 | 🔴 |

**引用 CASE-M6-002**：
> 杜邦分析告诉我们：ROE高可能源于效益好、效率高、或杠杆大三种原因。
> 只有前两者是企业真正的盈利能力，靠借钱堆出来的ROE不可持续。

---

### Step 4：经济利润判断——是否真正为股东创造价值

**核心问题**：ROIC是否高于WACC？企业是否在销毁股东财富？

**计算指标**（数据允许时执行，来自 CASE-M6-004）：
```
投资资本 = 有息负债 + 股东权益
ROIC = 税后营业净利润 / 投资资本
WACC = 有息负债/投资资本 × 债务成本 × (1-税率)
       + 股东权益/投资资本 × 行业平均ROE
经济利润 = (ROIC - WACC) × 投资资本
```

**判断规则**：

| 结论 | 条件 | 含义 |
|------|------|------|
| 🟢 真正赚钱 | ROIC > WACC | 为股东创造超额价值 |
| 🟡 勉强及格 | ROIC ≈ WACC | 覆盖资本成本，无超额回报 |
| 🔴 实质亏损 | ROIC < WACC | 账面盈利，经济亏损，销毁股东财富 |

**ROIC < WACC 时 → 引用 CASE-M6-004**：
> 净利润为正只是会计的及格线，不代表经济上真正赚钱。
> 刘洋公司2017-2018年账面盈利，但ROIC低于WACC——经济利润为负，在消耗股东财富。

---

### Step 5：综合评定

**整合 M1-M6 所有信号，输出最终结论**：

判断框架：
```
好公司 = 行业位置良好 + 资产质量健康 + 利润真实可持续
       + 现金流充裕 + 偿债能力稳健 + 战略执行有效 + ROIC > WACC
```
