# Skill: 行业背景分析（财报透视 — Module 1）

**知识来源**：《一本书读懂财报》肖星著，清华大学经管学院教授，2019年修订版
**案例库**：`references/case_library.md`（所有 CASE-M1-XXX 原文引用均来自此文件）
**独立使用**：本 skill 可单独调用，也可作为 `../../SKILL.md` 编排流程的第一步

## 调用时机

以下任一情况可单独调用本 skill：
- 用户想了解某行业的财务特征和竞争格局
- 用户想知道某项财务指标（毛利率、应收账款等）在该行业是否正常
- 用户想判断企业的战略定位（成本领先 vs 差异化）
- 作为完整财报分析的第一步，为后续资产、利润、现金流分析建立基准

---

## 模块输入

由 orchestrator 传入以下信息（有多少传多少，不足的在执行中追问用户）：
- 公司名称 / 股票代码
- 所在行业（若已知）
- 用户已提供的财务数据（毛利率、应收账款、固定资产等，若有）

## 模块输出

执行完成后，向 orchestrator 返回「行业坐标基准报告」（见 Step 5），
供后续模块使用以下字段：
- `行业类型标签`
- `各指标合理区间`（毛利率、应收账款、固定资产等）
- `上下游议价权初判`
- `战略定位初判`
- `Module 1 风险提示列表`

## 执行流程：行业背景分析（先于任何数字判断执行）

### 目的
在解读任何财务数字之前，先建立"行业坐标系"。
为后续模块提供：行业类型标签、各指标的合理区间、竞争格局摘要。

### 执行步骤

---

**Step 1：识别行业类型**

首先确认公司所在行业，然后从 `cases_library.md` 末尾的「行业类型参照表」中匹配对应类型。

询问用户（如果信息不足）：
- 这家公司主要做什么业务？卖给企业（to B）还是卖给消费者（to C）？
- 主营产品是实体产品还是服务/软件？
- 是否属于重资产制造业（需要大量设备厂房）？

**行业类型识别结果应输出**：
```
行业类型：[重资产制造 / 轻资产消费品 / to B 设备 / to C 零售 / 渠道依赖型 / 平台互联网 / 医药 / 房地产]
固定资产占总资产合理区间：[X% - Y%]
毛利率参考区间：[X% - Y%]
应收账款特征预期：[低/中/高，原因]
```

---

**Step 2：评估竞争强度（反映在毛利率）**

从以下维度判断行业竞争烈度：

**2a. 行业内部竞争**
- 该行业是否产能过剩？（供大于求 → 被迫降价 → 毛利率下行）
- 行业集中度如何？（CR4/CR5 越低，竞争越激烈）
- 是否存在技术或品牌壁垒？（有壁垒 → 有定价权 → 毛利率稳定）

**2b. 新进入者与替代品威胁**
- 该行业进入门槛高吗？（资本密集 → 壁垒高 → 毛利率相对稳定）
- 是否有替代产品正在颠覆市场？

**判断规则**：
- 行业竞争激烈 → 毛利率应偏低，若偏高需追问差异化来源
- 行业竞争温和 → 毛利率应偏高，若偏低需追问成本或定价问题

**⚡ 引用指令 → CASE-M1-001**：
当用户提出毛利率数字时，先读取 `cases_library.md` 中的 CASE-M1-001，
引用造纸（28%）vs 彩电（15%）的对比，向用户说明：
> 同一年代，不同行业的毛利率基准可以相差一倍，必须用行业基准评估，而非主观判断高低。

**⚡ 引用指令 → CASE-M1-002**：
若毛利率呈现连续多年下行，读取 CASE-M1-002，
引用造纸行业2002-2012年毛利率从28%→16%的案例，说明：
> 毛利率腰斩不一定是企业经营变差，可能是整个行业进入产能过剩阶段的必然结果。
> 需区分：是行业性下滑，还是该企业相对同行下滑更快？

---

**Step 3：判断上下游议价权（反映在应收账款和应付账款）**

**3a. 下游客户议价权（看应收账款）**

| 信号 | 应收账款/总资产 | 含义 |
|------|----------------|------|
| 下游弱势 | 低（<5%） | 企业定价权强，现款现货，客户没有赊账机会 |
| 下游均衡 | 中（5-15%） | 行业正常信用销售 |
| 下游强势 | 高（>15%） | 企业被迫赊销，客户话语权强 |

判断步骤：
1. 计算应收账款/总资产
2. 与行业参照表对比
3. 若高于参照区间 → 触发 CASE-M1-003 引用

**⚡ 引用指令 → CASE-M1-003**：
若应收账款占比高于行业参考值，读取 CASE-M1-003，
引用家电企业应收账款占23%、造纸企业占14%的案例，说明：
> 应收账款高不是销售旺盛的表现，往往是企业在竞争压力下，
> 被迫给下游客户提供赊账才能卖出货——是议价权弱的信号，而非业绩好的信号。

追问用户：
- 该公司的前五大客户集中度是多少？
- 是否存在单一大客户依赖？

**3b. 上游供应商议价权（看应付账款）**

| 信号 | 应付账款/总资产 | 含义 |
|------|----------------|------|
| 供应商弱势 | 高（>15%） | 企业可以延迟付款，免费占用供应商资金 |
| 供应商均衡 | 中（8-15%） | 正常商业信用 |
| 供应商强势 | 低（<8%） | 企业必须现款购货，无法占用供应商资金 |

**⚡ 引用指令 → CASE-M1-004**：
若大型企业应付账款占比低于同行或低于8%，读取 CASE-M1-004，
引用造纸行业龙头应付账款只有8%、反低于普通彩电企业15%的案例，说明：
> 规模大的企业应付账款反而低，不是因为财务保守，而是上游供应商强势——
> 造纸依赖进口木浆，供应商要求现款现货，企业议价权受限，成本压力持续存在。

追问用户：
- 该公司的核心原材料是否高度依赖特定供应商或进口？
- 原材料价格波动是否直接传导到成本端？

---

**Step 4：识别企业战略定位（为后续模块建立评估框架）**

在行业背景清晰后，初步判断企业采用哪种竞争战略：

**成本领先战略**（效率制胜）：
- 财务特征：毛利率低 + 总资产周转率高
- 适合行业：重资产制造业、规模经济显著的行业
- 验证方法：规模扩大时单位成本是否显著下降？

**差异化战略**（效益制胜）：
- 财务特征：毛利率高 + 总资产周转率低（高单价低销量）
- 适合行业：消费品品牌、精密制造、专业服务
- 验证方法：提价时销量是否基本稳定？

**战略执行失败信号**（两头不靠）：
- 声称成本领先，但周转率也低 → 规模没有带来效率优势
- 声称差异化，但毛利率也低 → 没有做出真正的差异

**⚡ 引用指令 → CASE-M1-006**：
判断战略定位时，读取 CASE-M1-006，
引用同属造纸行业的卷烟纸企业（差异化，高毛利低周转）vs 新闻纸企业（成本领先，低毛利高周转），
以及苹果（差异化，净利润率21.2%，周转率0.6次）vs 华为（均衡型，净利润率7.1%，周转率1.2次），
说明两种战略的财务特征是可以预期和验证的。

---

**Step 5：输出「行业坐标基准报告」**

完成以上步骤后，输出以下结构的报告，供用户确认，并作为后续模块的分析基础：

```
═══════════════════════════════════════════
  行业坐标基准报告 — [公司名称]
═══════════════════════════════════════════

【行业类型】
[类型名称]，[to B / to C]，[重/轻资产]

【竞争格局】
当前竞争强度：[激烈 / 温和 / 垄断性]
产能状态：[过剩 / 均衡 / 供不应求]
行业趋势：[扩张期 / 成熟期 / 衰退期]

【各指标合理区间（基于行业基准）】
毛利率合理区间：[X% - Y%]
应收账款/总资产合理区间：[X% - Y%]
应付账款/总资产合理区间：[X% - Y%]
固定资产/总资产合理区间：[X% - Y%]

【上下游议价权初判】
对上游供应商：[强势 / 均衡 / 弱势]，依据：[简述]
对下游客户：[强势 / 均衡 / 弱势]，依据：[简述]

【战略定位初判】
推测战略：[成本领先 / 差异化 / 待确认]
验证指标：毛利率 [X%]，总资产周转率 [X次/年]

【Module 1 风险提示】
[列出本模块发现的异常信号，如有]

【下一步】
请提供以下数据以继续分析：
- 最近3年毛利率、净利润率
- 应收账款、存货余额及营业收入
- 经营/投资/融资三项现金流
═══════════════════════════════════════════
```

## 案例引用操作规范

每次引用 `cases_library.md` 中的案例时，格式如下：

```
📖 原书案例：[案例标题]

[直接引用原文段落，保持原文表述]

这个案例说明的是：[一句话连接案例与当前公司的情况]
```

**规则**：
- 只引用与当前诊断结论直接相关的案例，不要堆砌引用
- 引用后必须说明"这个案例说明的是"，将抽象原则连接到用户具体的公司
- 如果案例与当前情况有差异，需要说明"与该案例的区别在于……"

## 不应做的事

- ❌ 不能在没有行业背景的情况下评价毛利率高低
- ❌ 不能把应收账款高简单等同于"销售好"
- ❌ 不能把固定资产占比高简单等同于"资产沉重有风险"
- ❌ 不能跨行业横向比较毛利率
- ❌ 不能只看一年数据，必须追问3年趋势
- ❌ 不能给出"这家公司好/不好"的结论，只能给出"某维度存在/不存在风险信号"
