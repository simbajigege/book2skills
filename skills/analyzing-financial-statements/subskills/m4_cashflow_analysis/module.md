# Skill: 现金流健康度（财报透视 — Module 4）

**知识来源**：《一本书读懂财报》肖星著，清华大学经管学院教授，2019年修订版
**案例库**：`references/case_library.md`（所有 CASE-M4-XXX 原文引用均来自此文件）
**独立使用**：本 skill 可单独调用，也可接收 M1/M3 报告后作为编排流程第四步

## 调用时机

以下任一情况可单独调用本 skill：
- 用户想判断某公司现金流是否健康、能否持续经营
- 用户发现盈利但现金流为负，或现金流改善但利润下滑
- 用户想评估企业扩张投资的风险
- 用户询问企业处于哪个生命周期阶段

---

## 模块输入

- M1 行业基准报告（行业现金流特征、扩张节奏）
- M3 利润质量报告（利润与现金流差异分析背景）
- 用户提供的现金流量表：经营/投资/融资活动现金流，净现金流
- 连续2-3年数据最佳

## 模块输出

向 orchestrator 返回「现金流健康报告」，包含：
- 8类企业分类定位
- 主要风险来源判断
- 投资节奏与资金链安全评估

---

## 执行流程

### Step 1：8类现金流分类定位

**判断逻辑**：

```
经营现金流正负 → 投资现金流正负 → 融资现金流正负 → 确定类型（1-8类）
```

参考 **CASE-M4-001** 的8类框架表，确定企业当前所处类型。

**输出示例**：
> 该公司经营+、投资-、融资+，属于**第三类（成长扩张型）**。
> 核心风险：投资节奏与方向是否可控。

---

### Step 2：经营现金流质量

**核心问题**：经营活动是否具备造血功能？现金含量是否充足？

**计算指标**：
```
现金含量 = 经营活动现金流 / 净利润
经营现金流趋势（近3年）
经营现金流 vs 投资支出比
```

**判断规则**：

| 信号强度 | 条件 | 评级 |
|---------|------|------|
| 🟢 健康 | 经营现金流持续为正，现金含量 ≥ 0.8 | 造血功能良好 |
| 🟡 关注 | 经营现金流为正但下降趋势，现金含量 0.5-0.8 | 回款效率下降 |
| 🔴 风险 | 经营现金流为负，或现金含量持续 < 0.5 | 主业无法自给自足 |

**经营现金流为负时 → 引用 CASE-M4-004**：
> 利润通常比现金流更重要——但当资产面临减值风险时，现金流量表才是最后的防线。

---

### Step 3：投资活动分析

**核心问题**：投资方向是否清晰？投资规模是否超出企业承受能力？

**计算指标**：
```
投资覆盖率 = 经营活动现金流 / 投资活动现金流流出
净现金流 = 经营 + 投资 + 融资现金流之和
```

**判断规则**：

| 信号强度 | 条件 | 评级 |
|---------|------|------|
| 🟢 健康 | 投资方向聚焦主业，经营现金流覆盖投资支出50%以上 | 扩张节奏合理 |
| 🟡 关注 | 投资规模大，需大量融资支撑，但主业良好 | 关注融资节奏 |
| 🔴 风险 | 投资规模远超经营+融资能力，净现金流持续大幅为负 | 资金链断裂风险 |

**扩张过猛时 → 引用 CASE-M4-002**：
> 方便面企业的教训：主业稳健、产品热销，但投资扩产太快。
> 法院查封时账上只剩3000元——不是生意不好，是现金流撑不住投资节奏。

---

### Step 4：融资活动与净现金流

**核心问题**：企业的净现金流是否安全？融资方式是否可持续？

**计算指标**：
```
净现金流 = 经营 + 投资 + 融资
融资依赖度 = 融资现金流 / 投资活动流出
```

**判断规则**：

| 信号强度 | 条件 | 评级 |
|---------|------|------|
| 🟢 健康 | 净现金流为正，或融资来源稳定银行授信 | 资金链安全 |
| 🟡 关注 | 净现金流为负但在可控范围，依赖银行短贷 | 需关注借款续借能力 |
| 🔴 风险 | 净现金流大幅为负，且需依赖信托/高息融资 | 资金成本失控，危机预警 |

**经营现金流转负、依赖高息融资时 → 引用 CASE-M4-003**：
> 2008年房地产企业的警示：经营现金流一旦转负，第四类瞬间变第八类。
> 七折卖房、高息信托借款，都是被逼之举——宏观敏感行业需要更多现金储备。

---

### Step 5：利润 vs 现金流矛盾判断

**核心问题**：利润和现金流出现矛盾时，该信哪个？

**判断框架**（来自 CASE-M4-004）：

```
资产质量可靠（大客户、有订单支撑）→ 信利润表
资产减值风险高（卖不掉的存货、收不回的账款）→ 信现金流量表
```

**操作步骤**：
1. 确认利润与现金流差异的原因（应收账款增加？存货增加？）
2. 评估对应资产的回收概率
3. 输出判断：利润可信 / 现金流更重要
