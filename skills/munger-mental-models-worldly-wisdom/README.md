# Munger Mental Models and Worldly Wisdom

Apply Charlie Munger's latticework of mental models, inversion, circle-of-competence discipline, and misjudgment checks to investments and hard decisions.

## What It Does

- Maps complex decisions across multiple disciplines instead of one narrow lens.
- Runs investment reviews with competence, inversion, risk, opportunity cost, and bias checks.
- Diagnoses psychological misjudgment and lollapalooza effects in markets, organizations, and personal decisions.
- Adds citation rules tied to the Poor Charlie's Almanack quote library.

## When to Use It

- Analyze this company using Munger's approach.
- What mental models apply here?
- Spot the biases in this decision.
- Invert this plan before I act.
