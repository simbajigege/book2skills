# Common Sense Index Investing (Bogle)

A Claude skill distilled from John C. Bogle's *The Little Book of Common Sense Investing*. It applies Bogle's index fund framework to help investors evaluate fund choices, understand the tyranny of costs, set stock/bond allocation, and resist performance-chasing — all grounded in the arithmetic of compounding.

## What It Does

- Evaluates any mutual fund or ETF using Bogle's cost-first framework
- Computes the long-term wealth destruction caused by expense ratios and turnover costs
- Applies the reversion-to-mean test to funds selected on past performance
- Recommends stock/bond allocation based on time horizon and risk tolerance
- Distinguishes business investment returns (reliable) from speculative returns (unpredictable)

## When to Use It

Trigger this skill when a user asks:
- "Should I use index funds or actively managed funds?"
- "Is this fund's expense ratio reasonable?"
- "How should I split my portfolio between stocks and bonds?"
- "This fund had great returns last year — should I buy it?"
- "Is now a good time to invest?"
