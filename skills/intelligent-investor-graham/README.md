# Intelligent Investor — Graham Value Investing Skill

A Claude skill that encodes Benjamin Graham's framework from *The Intelligent Investor* (Revised Edition). It helps individual investors make disciplined, analysis-driven decisions about stock valuation, portfolio allocation, and market psychology — without speculation.

## What It Does

- **Evaluates stocks** against Graham's 7 quantitative criteria for defensive investors (size, financial strength, earnings stability, dividend record, earnings growth, P/E, price-to-book)
- **Calculates margin of safety** — the gap between price paid and estimated intrinsic value
- **Classifies operations** as investment or speculation based on Graham's definition
- **Applies the Mr. Market mental model** to reframe market declines and price swings as opportunities, not threats
- **Recommends portfolio allocation** between stocks and bonds using the 25/50/75 system

## When to Use It

Trigger phrases that activate this skill:

- "Is this stock worth buying at this price?"
- "Is this investment or speculation?"
- "The market dropped — should I sell?"
- "How should I allocate my portfolio between stocks and bonds?"
- "Does this company qualify as a defensive investment?"
- "What is the margin of safety on this stock?"
- "Evaluate [company] using Graham's criteria"

## Installation

1. Download or clone this folder.
2. In Claude.ai, go to **Settings → Skills** and upload the skill folder.
3. The skill will appear in your available skills list.
