# Intelligent Investor Graham

Apply Benjamin Graham's value investing framework from *The Intelligent Investor* to stock screening, margin of safety analysis, market fluctuation decisions, portfolio allocation, and investment-versus-speculation classification.

## What It Does

- Screens stocks with Graham's defensive investor criteria.
- Estimates whether a current price offers a margin of safety.
- Reframes volatility through the Mr. Market mental model.
- Guides stock/bond allocation within Graham's defensive guardrails.

## When to Use It

Use this skill for questions like "Is this stock worth buying?", "Does this company pass Graham's defensive criteria?", "What price would provide a margin of safety?", "Should I sell after a market drop?", or "Is this trade investment or speculation?"

## Installation

### Option 1 — CLI (recommended)

```bash
npx skills add simbajigege/book2skills/skills/intelligent-investor-graham
```

### Option 2 — Manual upload

1. Download the skill folder (or clone this repo).
2. In Claude.ai, go to **Settings -> Skills** and upload the folder.
3. The skill will appear in your available skills list.
