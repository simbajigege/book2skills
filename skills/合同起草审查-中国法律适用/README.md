# 合同起草审查（中国法律适用）

A structured contract drafting and review skill based on Chinese law (民法典 / Civil Code 2021). It applies the **三观四步法 (Three-Perspective Four-Step Method)** to help lawyers, legal counsel, and law students systematically draft and review contracts governed by the laws of the People's Republic of China.

## What It Does

- **Macro analysis (宏观)**: Evaluates transaction structure — contract type, parties, subject matter, and legal procedures — to determine whether a deal is feasible and how it should be structured.
- **Intermediate analysis (中观)**: Selects the appropriate contract form (LOI, pre-contract, framework agreement, electronic contract, etc.).
- **Micro analysis (微观)**: Drafts and reviews specific contract clauses following the "Three Points One Line" (三点一线) structure.
- **Full workflow support**: Guides users through the four-step process — requirements gathering → three-perspective analysis → verification → submission.

## When to Use It

Trigger phrases and scenarios:
- "审查这份合同" / "帮我看一下这个合同"
- "起草一份合同" / "合同草稿怎么写"
- "这笔交易有什么风险" / "合同条款有问题吗"
- "交易结构怎么设计" / "这个条款合不合适"
- Reviewing purchase agreements, service contracts, lease agreements, cooperation agreements, etc. under Chinese law
