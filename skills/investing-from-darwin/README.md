# What I Learned About Investing from Darwin

This skill distills *What I Learned About Investing from Darwin* by Pulak Prasad into an agent-ready decision-support workflow for investing, evolution, risk control.

## What It Does

- Applies the book's core framework to practical user questions.
- Provides checklists, routing rules, warning signs, and output formats.
- Uses quote anchors for source-grounded citations.
- Keeps source passages in `quotes/` rather than workflow prose.

## When to Use It

Use this skill for questions involving investing, evolution, risk control, quality companies, especially when the user wants a structured diagnosis, critique, or decision framework.

## Installation

### Option 1 — CLI (recommended)

```bash
npx skills add simbajigege/book2skills/skills/investing-from-darwin
```

### Option 2 — Manual upload

1. Download the skill folder or clone this repo.
2. In Claude.ai, go to Settings -> Skills and upload the folder.
3. The skill will appear in your available skills list.
